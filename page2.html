Requirement 1

The micro-controller shall control two stepper motors:

  • Horizontal Motor controlled by Led 1, Led 2, Led 3, and Led 4a
  • Vertical Motor controlled by Led 5, Led 6, Led 7, and Led 8

Requirement 2

Switch 1 will control Stop/Run functionality. This should turn the system off immediately.

  • 0: Stop
  • 1: Run

Requirement 3

Switch 2 will control Manual/Auto mode

  • 0: Manual Mode - The laser will point to the following
  • 1: Auto Mode - The laser pointer performs the following movements
    • Move to C, wait 3 seconds
    • Move to B, wait 3 seconds
    • Move to A, wait 3 seconds
    • Move to Home, wait 3 seconds
    • Repeat the above sequence continuously until interrupted

Additional Features

When the above requirements are complete and tested to be fully working, these additional features can be implemented

Real Laser Control

  • This would involve 2 buttons, BTN1 and BTN2 for the X and Y axis, and the POT.
  • Each button will have 3 states
    • Off - default
    • Editing - adjusting target using POT, if both BTNs are in this state, allow adjusting both numbers at once
    • Active - When both BTNs are in this state, the laser will begin to move to the target, otherwise be doing what switch 2 is
  • Use the Oled to show the necessary information to the user

Shoot Laser

  • Designated to button 3, this will make the laser blink to simulate it "shooting" upon a click

One Extra Button

  • Crazy idea - when button 4 is clicked, a simple game appears on the Oled that can be played by the player
  • Ok idea - Allow changing of the manual and automatic X,Y coordinates
  • Boring idea - credits scroll through the Oled
