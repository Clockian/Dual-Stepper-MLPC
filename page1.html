Deliverables

  1. Project folder containing all files required to run system operation with source code comments
  2. Video to prove that the system works
    • Maximum of 1 minute long
    • MP4 format
  3. One Shot demonstration of system
  4. Powerpoint containing the following
    • Project Title with team members and date
    • Project Description
    • System Block Diagram
    • Functional Requirements
    • Development Platform Description
    • Control Flow Chart
    • Design Features/Sample Code
    • Problems encountered and associated solutions
    • Developer's Notes at the end
